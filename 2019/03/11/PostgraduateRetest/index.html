自我介绍（中文）

老师们好，我是柯锐，我是一名来自湖北省黄石市阳新县的农村孩子。一家有六口人，爸爸妈妈在外省工作。我们家是一个普普通通的农村家庭，爸爸妈妈不会给我讲些大道理，但是他们潜移默化的教会了我做一个谦卑、积极进取的人。因此，大学四年里我没有荒废自己的时间。大一，我参加了吉他社，骑行社，偶尔会练练吉他陶冶下自己的心情。假期，我会和学校的骑行队骑到北京天津等地方，骑行需要毅力，因为要赶路，不能因为自己一个人拖慢了整个车队的进程。骑行也需要团队协作能力，因为我们一个队伍有十几个人，如果队伍中有一个人不小心翻倒了，可能导致整个队伍发生追尾，这在马路上是很危险的，所以我们车队会有个领队，还有一个负责在车队的走后面。我们每个人都要时刻关心着自己其后的队友，从出发地到目的地往往需要半天甚至一天的时间。骑行锻炼了我的毅力和团队协作能力。大二，我加入了计算机学院的创新实验室，因为我想增强自己的专业知识实践能力。在实验室里我可以和老师和师兄师姐一起做些竞赛，大二下学期，我参加了全国大学生智能车竞赛，我们团队获得了华北赛区一等奖，全国二等奖。我还参见了Mathorcup数学建模，并获得了二等奖，参加了中国高校风控管理挑战赛，最终取得了特等奖。大三，我参加了华北五省机器人大赛，获得了河北赛区二等奖。到了大四，越学越觉得自己的无知，于是想进一步的提高自己的专业能力以及思想高度，于是决定考研。大学四年我拿了六次一等奖学金，获得了三好学生，优秀团员，优秀团干部等称号，现在是一名预备党员，今年六月份将要转为正式党员。我的大学四年虽然没有什么风风光光的成就，但是我脚踏实地，大学生活过的很充实。我很期待我的研究生生活，研究生阶段我会继续脚踏实地，进一步提高自己的专业能力，并希望在导师的指导下，今后在专业领域能够取得一定成就。

个人简历

项目经历：

  • 智能车图像处理：硬件平台是K60，对智能车上的摄像头采集的图像，通过设计算法进行物体识别，并对智能车进行路径规划。主要使用的编程语言为C，期间会用到matlab。
  • 机器人编程：硬件平台是stm32, 通过机器人身上传感器采集的信息，进行分析，然后针对不同情况，对机器人发出不同的控制行为，主要是控制电机的转速，实现机器人的前进，后退，左转，右转，加速，减速等。
  • 个人网站：网站代码托管到了GitHub上，有自己的域名（www.keruer.com），网站主要的用途是写写文章，发发博客，总结自己的经历，记录自己的学习笔记。
  • 电子音乐盒：硬件平台是51单片机，通过按键可以切换歌曲。
  • 电子遥控风扇：硬件平台是51单片机，通过红外遥控器，可以控制不同的风速。
  • 深度学习手写字体识别: 使用谷歌的开源框架Tensorflow进行手写字体识别。

获奖情况：

  • 全国大学生智能车竞赛全国二等奖，华北赛区一等奖。
  • 华北五省机器人大赛河北赛区二等奖
  • Mathorcup大学生数学建模二等奖
  • 中国高校风控管理挑战赛特等奖
  • 一等奖学金六次

获得的荣誉称号：

  • 三好学生
  • 优秀团干部
  • 优秀团员
